## Objetivo

Aprofundar o desenvolvimento de sistemas web utilizando:
- Linguagem de programação para páginas dinâmicas
- Comunicação síncrona e assíncrona
- Mapeamento objeto-relacional (ORM)
- Padrões de projeto aplicados à web

---

## Ementa

- Frameworks de desenvolvimento web    
- Padrões de projeto e Programação Orientada a Objetos
- Chamadas assíncronas (AJAX)
- Aplicações web em n-camadas
- Conexão com banco de dados utilizando ORM
- Manipulação de requisições GET e POST
- Sessão e cookies
- Serialização de objetos
